# Outcome KPIs — sharper-copy

| KPI | Target | Measurement method | Baseline |
|-----|--------|-------------------|----------|
| Hero scroll-through rate | +15% increase in visitors scrolling past hero | Umami: scroll depth event or time-on-page for hero section | Current (pre-revision) scroll rate |
| Project CTA click-through rate | +10% increase in project-cta umami events | Umami: `project-cta` events per session | Current CTA click rate |
| Note click-through rate | +10% increase in note-click umami events | Umami: `note-click` events per session | Current note click rate |
| Bounce rate on homepage | -5% decrease | Umami: bounce rate on `/` | Current bounce rate |
| 3-second test (qualitative) | ≥3 of 5 builders describe the site as "civic tools in Cluj" not "software engineer" | Manual test: ask 5 builders to read hero and describe what you do | Current: likely "software engineer" |
