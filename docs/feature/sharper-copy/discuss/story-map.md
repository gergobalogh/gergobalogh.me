# Story Map — sharper-copy

## Backbone (activities, left to right)

1. **Hero** — First 3 seconds determine everything
2. **About** — Deepen the signal for those who stay
3. **Projects** — Confirm the promise with concrete work
4. **Notes** — Prove the thinking is real
5. **Project detail pages** — Close the loop for those who clicked

## Walking Skeleton

**Slice 0 (Hero):** Rewrite hero headline + subhead in EN. Ship it. Validate the 3-second test with real visitors.

## Elephant Carpaccio Slices

| Slice | What ships | End-to-end value | ≤1 day? | Learning hypothesis |
|-------|-----------|-------------------|---------|---------------------|
| S1 | Hero rewrite (EN) | Visitor reads hero and can describe what you do in one sentence | Yes | "If the hero frames the work specifically, builders will scroll past instead of leaving" |
| S2 | Hero rewrite (RO + HU) | All-language visitors get the same 3-second test | Yes | "RO/HU translations preserve the sharp framing, not just literal meaning" |
| S3 | About restructure (EN + RO + HU) | "Build for use, not demonstration" lands before visitor scrolls past | Yes | "Thesis-first About retains collaborators who currently bounce at LinkedIn-style copy" |
| S4 | Project card reframe — isthisai (all langs) | isthisai card makes the reader think "I want to see this" | Yes | "Reframed description increases click-through from project cards" |
| S5 | Project card reframe — clujparticipa (all langs) | clujparticipa CTA invites participation, not display | Yes | "Action-oriented CTA increases click-through on clujparticipa" |
| S6 | Project card reframe — notes-and-experiments + status labels (all langs) | Notes card framed correctly; status signals life | Yes | "Non-Jira status labels make in-progress projects feel active" |
| S7 | Notes intro rewrite (all langs) | Notes section hooks instead of categorizing | Yes | "A hook intro increases click-through to individual notes" |

## Prioritization

1. **S1** — highest learning leverage. If the hero fails, nothing else matters.
2. **S2** — dependency: EN hero must land before localizing.
3. **S3** — About is the gate for JOB-2 (collaborators). High impact.
4. **S4–S5** — project cards confirm the hero's promise. Medium impact.
5. **S6** — status labels + notes card. Lower impact but closes the loop.
6. **S7** — notes intro. Lowest urgency but completes the arc.
