# Slice 01: Hero rewrite (EN)

## Goal
Visitor reads the hero and can describe what you do in one sentence that is NOT "software engineer."

## IN scope
- Rewrite `heroHeadline` in English
- Rewrite `heroSubhead` in English
- Update `siteDescription` meta tag in English

## OUT scope
- RO/HU translations
- About section changes
- Project card changes
- Notes section changes
- Any layout or component changes

## Learning hypothesis
If the hero frames the work specifically (not categorically), builders will scroll past instead of leaving. This is disproved if analytics show no change in scroll depth or time-on-page for the hero section.

## Acceptance criteria
- Hero headline passes the 3-second test: a builder reads it and either clicks through or leaves thinking "civic tools in Cluj" — not "software engineer"
- Hero subhead deepens the signal without repeating the headline
- Hero subhead preserves the current three-part structure ("see more clearly, take part directly, act on what they know") or improves it
- No new layout changes — copy only

## Dependencies
None. This is the walking skeleton.

## Effort estimate
~2 hours (copy writing + review)
