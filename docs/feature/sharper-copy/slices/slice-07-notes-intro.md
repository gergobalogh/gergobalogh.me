# Slice 07: Notes intro rewrite (all languages)

## Goal
The notes section intro hooks the reader, not categorizes the content.

## IN scope
- Rewrite `notesIntro` in EN, RO, HU (homepage notes section)
- Rewrite `notesPageIntro` in EN, RO, HU (notes index page)

## OUT scope
- Changes to individual note titles or summaries
- Changes to any other section

## Learning hypothesis
A hook intro increases click-through to individual notes. Disproved if analytics show no change in note click rate.

## Acceptance criteria
- The intro creates urgency ("these might change how I think about X") not a category ("notes about X")
- It preserves the current subjects (civic software, publishing, building) but frames them as ideas-in-progress
- RO and HU translations preserve the hook framing

## Dependencies
- S1 (hero sets the tone; notes intro should match)

## Effort estimate
~1 hour (3 languages, 2 fields each)
