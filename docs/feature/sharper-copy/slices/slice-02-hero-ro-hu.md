# Slice 02: Hero rewrite (RO + HU)

## Goal
Romanian and Hungarian visitors get the same sharp 3-second test as English visitors.

## IN scope
- Rewrite `heroHeadline` in RO and HU
- Rewrite `heroSubhead` in RO and HU
- Update `siteDescription` in RO and HU

## OUT scope
- Changes to EN copy
- Any other section changes

## Learning hypothesis
RO/HU translations can preserve the sharp framing (not just literal translation). This is disproved if native speakers report the translations feel flat or generic compared to EN.

## Acceptance criteria
- RO hero headline passes the 3-second test for a Romanian-speaking builder
- HU hero headline passes the 3-second test for a Hungarian-speaking builder
- Neither translation is a literal word-for-word of EN — they adapt the framing to the target language
- Both preserve the "specificity over category" principle

## Dependencies
- S1 (EN hero must land first to establish the pattern)

## Effort estimate
~2 hours (translation + review)
