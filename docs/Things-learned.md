## 2026-05-20

**Literal translation is the fastest way to make multilingual copy sound wrong.** Each language has its own rhetorical devices — use them, or the copy will read like English with foreign words.

The clujparticipa description went through this exact failure. The English version uses a challenge-led question ("What does the city want before formal politics decides?"). Translating that structure into Hungarian produced something that technically meant the same thing but felt like translated logic, not Hungarian. The fix was a rhetorical flip question native to Hungarian: "Mi lenne, ha megfordítanánk?" — "What if we flipped it?" Same challenge framing, different device. Romanian went its own way entirely: "Bugetarea participativă a murit la Cluj" — a locally grounded statement about Cluj's specific history, not a question at all. Three languages, three structures, one intent.

**Status labels on a personal site should signal life, not ticket state.** "In progress" and "Ongoing" read like Jira fields — they tell you a process exists, not that something is alive.

The fix: replace process-jargon with signals that convey momentum or presence. "Under construction" says someone is actively building. "Always shipping" says output keeps coming. "In operation" says this thing is out there working. The pattern: ask what a visitor should *feel* about the project's state, then pick the word that produces that feeling.

**When a user corrects your copy, stop generating options and listen.** The best RO and HU copy in this session came from the user, not from option lists.

The pattern repeated twice: I'd offer 3-4 variations, the user would reject all of them, then write something sharp themselves. The Hungarian description ("A város dönt, mielőtt kérdezne. Mi lenne, ha megfordítanánk?"), the Romanian description ("Bugetarea participativă a murit la Cluj"), the RO notes CTA ("Idei în lucru") — all user-authored. The lesson: when option lists keep missing, the problem isn't selection, it's that I'm writing from English logic. Stop, ask what we're trying to say, and let the native speaker find the device.

**"Something fucking simple" beats clever framing.** The RO notes CTA went through workshop metaphors, draft-reading invitations, and process-following — all overengineered. The user cut through: "Idei în lucru" — "ideas in progress." Two words, humble, human. When copy feels overwrought, strip it back to what the thing simply is.
